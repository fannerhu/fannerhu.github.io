View

rotate

icon:

Use drag gesture to rotate around the X and Y axes. Use magnify gesture to rotate around the Z axis.
Set rotation: Long press the main button to show the input fields. Input the rotation value for each axis. The unit is degree here. Then choose the mode. "Total" mode means the rotation is performed from the initital rotationless state. "Added" mode means the rotation is perfromed from current state.

transit

icon:

Use drag gesture to transit along the X and Y axes. Use magnify gesture to transit along the Z axis.
Set transition: Long press the main button to show the input fields. Input the transition value for each axis.

zoom

icon:

Use magnify gesture to zoom in and out.

render quality setting

Three qualities to select: low, middle, high.
