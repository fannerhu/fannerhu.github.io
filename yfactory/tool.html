Tool

clip

icon:

Use drag gesture to perform clip operation. The smaller part will be removed and the bigger part will be left.

boolean

icon:

Preform boolean operations between model A and model B.

open
icon:

Open model B. It could be a primitive model or self-created model.

rotate
icon:

Rotate model B. Use drag gesture to rotate model B around the X and Y axes. Use magnify gesture to rotate model B around the Z axis.
Set rotation: Long press the main button to show the input fields. Input the rotation value for each axis. The unit is degree here. Then choose the mode. "Total" mode means the rotation is performed from the initital rotationless state. "Added" mode means the rotation is perfromed from current state.

transit
icon:

Transit model B. Use drag gesture to transit model B along the X and Y axes. Use magnify gesture to transit model B along the Z axis.
Set transition: Long press the main button to show the input fields. Input the transition value for each axis.

scale
icon:

Scale model B. Use magnify gesture to scale it bigger or smaller.
Set Scale: Long press the main button to show the input field. Input the scale value.

setting
icon:

If model B is a primitive model, its dimension can be set here.
Set boolean mode: Union or Difference.

complete
icon:

Complete the boolean operation.
