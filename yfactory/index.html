yfactory Documents

yfactory is an easy-to-use iOS application for constructing 3D models.
It can work without a stylus pen.
It can work without Internet connection.
And it is free.

Two ways to navigate the view: use gestures and input values.
Three render qualities selection. Choose render quality based on the device.
Use drag gesture to clip the model.
Use Boolean operation to build from simple to complex.
Exporting to obj format is supported.
You can install it from here.
